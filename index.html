レポート作成手順

  1. レポート(テキストファイル)とプログラムのソースコードを(単一のディレクトリに)まとめたzipファイルを作成
  2. 下のフォームに従って提出

注意点

  • 提出するファイルはインタプリタ, 型推論でそれぞれ1つずつ(必修・オプション課題毎に提出するのではない)
  • 不備に自分で気づいた場合はこのフォームを使って再度提出すること
  • ソースコードはmakeするのに必要なファイルをすべて含むこと
  • ファイル投稿欄に記載されたインタフェース仕様を満たすこと
  • 不明な点はTAに問い合わせること

      ファイルを選択して下さい

      ソースファイルは以下のインタフェース仕様を満たすこと
      (ocamlc -i eval.ml等で確認できる)
      • Parsing.toplevel : (Lexing.lexbuf -> token) -> Lexing.lexbuf -> Syntax.program
      • Eval.eval_decl : dnval Environment.t -> Syntax.program -> ... * dnval Environment.t * ...
        (ただし, 返り値型のdnval Environment.tは2番目)
      • Eval.eval_exp : dnval Environment.t -> Syntax.exp -> dnval
      • Eval.exval = IntV of int | BoolV of bool | ProcV of ... | ListV of exval list
        (ただしListVはEx.3.16を解いた場合のみ)
      (ocamlc -i typing.ml等で確認できる)
      • Parsing.toplevel : (Lexing.lexbuf -> token) -> Lexing.lexbuf -> Syntax.program
      • Typing.ty_decl : Syntax.ty Environment.t -> Syntax.program -> ...
        ただし, Ex.4.9を解いた場合は
        Typing.ty_decl : Syntax.tysc Environment.t -> Syntax.program -> Syntax.tysc Environment.t * ...
      • Typing.ty_exp : Syntax.ty Environment.t -> Syntax.exp -> ... * Syntax.ty
        ただし, Ex.4.9を解いた場合は
        Typing.ty_exp : Syntax.tysc Environment.t -> Syntax.exp -> ... * Syntax.ty
      • Syntax.ty = TyInt | TyBool | TyVar of tyvar | TyFun of ty * ty | TyList of ty
        (ただしTyListはEx.4.8を解いた場合のみ)
